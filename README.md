# Небольшой телеграм бот для скачивания видео и звука из YouTube
## Для инсталляции
Библиотека телеграм бота aiogram (нужна именно 2.14)
```commandline
pip install aiogram=2.14
```
Библиотека для скачивания звука и видео из YouTube
```commandline
pip install yt-dlp
```
## Как работает
1. В bot_main.py вместо "BOT_TOKEN" вписываем токен бота
2. Запускаем bot_main.py
3. В телеграме отправляем /start
4. Выбираем /video или /audio
5. Отправляем ссылку на видео
6. Ссылка передаётся в download_video_audio.py и скачивется файл
7. Файл отправляется в чат
8. Файл удаляется из директории бота
9. Отправляем ещё ссылку или меняем режим скачивания
